<h3>Der Code des Jass-Rechners ist Open Source.
Jeder darf ihn benutzen, verändern und weiterverbreiten.</h3>
<br>
<h4>Dieses Projekt steht unter der GNU GPL v3 Lizenz.</h4>
<br>
<p>Für den auf android installieren muss man Termux installieren und das ein geben:</p>
<br>
<p>Fürs wget zu installieren:</p>
<pre>pkg update && pkg install wget -y </pre>
<br>
<p>Danach den install Datei laden und starten</p>
<pre>wget https://raw.githubusercontent.com/Andreaswinter2012/Jass-source-code/refs/heads/main/install && chmod +x install && ./install</pre>
<br>
<p>Zum starten den jass rechner in Termux das eingeben:</p>
<pre>python3 jass.py</pre>
<br>
<p>Und Auto start in Termux macht so:</p>
<pre>echo "python3 jass-mobil.py" >> /data/data/com.termux/files/usr/etc/bash.bashrc</pre>
<br>
<a href="">Exe Datei Windows 10/11 herunterladen von Google Drive</a>
<br>
<br>
<p>Damit man es mit Webbrowser start muss man es mit cmd öffnen</p>
<pre>jass.exe web</pre>
<br>
<a href="https://github.com/Andreaswinter2012/Jass-source-code/raw/refs/heads/main/jass.AppImage">AppImage Datei fürs Linux-systeme</a>
<br>
<br>
<p>Fürs den Datei zugriff geben das eingeben:</p>
<pre>chmod +x jass.AppImage</pre>
<br>
<p>Starten es durch:</p>
<pre>./jass.AppImage</pre>
<br>
<p>Oder mit Webbrowser:</p>
<pre>./jass.AppImage web</pre>
